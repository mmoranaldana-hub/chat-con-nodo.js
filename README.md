
Chat Master - Node.js with auth, SQLite, and Socket.io
=====================================================

Contenido generado en: /mnt/data/chat_project_node

Instrucciones rápidas:
1. cd /mnt/data/chat_project_node
2. npm install
3. npm start
4. Abre http://localhost:3000

Características añadidas:
- Registro e inicio de sesión (usuarios con contraseña hasheada)
- Sesiones persistentes usando SQLite store
- Mensajes guardados en SQLite (data/chat.db)
- Emisión en tiempo real con Socket.io
- Frontend SPA que maneja auth y chat

Notas:
- Cambia la variable de entorno SESSION_SECRET para producción.
- Para desplegar en Railway: sube este repo a GitHub y conecta Railway. Railway provee PORT automáticamente.
